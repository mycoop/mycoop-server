Copy/paste text passages, undo/redo your actions

To perform these operations, use the corresponding icons at the top toolbar:

  • Copy – select a text fragment and use the Copy icon to copy the selection to the computer clipboard memory. The copied text can be later inserted to another place in the same document, into another document, or into some other program.
  • Paste – find the place in your document where you need to paste the previously copied text fragment and use the Paste icon. The text will be inserted at the current cursor position. The text can be previously copied from the same document, from another document or from some other program.
  • Undo – use the Undo icon to undo the last operation you performed.
  • Redo – use the Redo icon to redo the last undone operation.

Note: for the security reasons some browsers don't allow access to your computer clipboard. So when you try to select one of the cutting operations, the program will ask you to use the keyboard shortcuts common for all browsers: Ctrl+X for cutting, Ctrl+C for copying, Ctrl+V for pasting.
