Establezca tipo,tamaño y color de letra

Usted puede seleccionar tipo, tamaño y color de la letra usando los iconos correspondientes que se sitúan en la barra de herramientas superior.

Nota: Si usted quiere aplicar el formato al texto existente ya en documento, selecciónelo con el ratón o use el teclado y aplique el formato.

Nombre de letra Se usa para elegir una letra de la lista de letras disponibles.
Tamaño de letra Se usa para elegir un tamaño de la letra del menú desplegable, o se puede introducirlo a mano en el campo de tamaño de letra.
Color de resaltado Se usa para cambiar el fondo de algunas oraciones, frases, palabras o carácteres. Pulse la flecha hacia abajo junto al icono para seleccionar el color. Para eliminar color de fondo, seleccione la opción Sin relleno. Color de resaltado se diferencia de Color de fondo porque el segundo se aplica a todo el párrafo.
Color de letra Se usa para cambiar el color de las letras/carácteres en el texto. Pulse la flecha hacia abajo junto al icono para seleccionar el color en las paletas disponibles. Los colores en paletas dependen de la combinación de colores seleccionada. Puede elegirla pulsando el icono Cambiar combinación de colores en la barra de herramientas superior.
