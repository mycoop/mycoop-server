Дополнительные параметры ONLYOFFICE™ Document Editor

Онлайн-редактор ONLYOFFICE™ Document Editor позволяет изменить дополнительные параметры. Для перехода к ним щелкните по значку Файл на левой боковой панели и выберите опцию Дополнительные параметры.... Можно также использовать значок , расположенный в правом верхнем углу верхней панели инструментов.

Доступны следующие дополнительные параметры:

  • Альтернативный ввод - используется для включения/отключения иероглифов.
  • Комментирование в реальном времени - используется для включения/отключения опции комментирования в реальном времени. Если отключить эту функцию, прокомментированные фрагменты будут подсвечиваться, только когда Вы нажмете на значок Комментарии .
  • Стандартное значение масштаба - используется для установки стандартного значения масштаба путем его выбора из списка доступных вариантов от 50% до 200%.
  • Отображать изменения при совместной работе - используется для определения изменений, которые необходимо подсвечивать во время совместного редактирования:
    • При выборе опции Все будут подсвечиваться все изменения, внесенные за время текущей сессии.
    • При выборе опции Последние будут подсвечиваться только те изменения, которые были внесены с момента, когда Вы последний раз нажимали на значок Сохранить .
  • Хинтинг шрифтов - используется для выбора типа отображения шрифта в ONLYOFFICE™ Document Editor:
    • Выберите опцию Как Windows, если Вам нравится, как обычно отображаются шрифты в операционной системе Windows, то есть с использованием хинтинга шрифтов Windows.
    • Выберите опцию Как OS X, если Вам нравится, как обычно отображаются шрифты в операционной системе Mac, то есть вообще без хинтинга шрифтов.
    • Выберите опцию Собственный, если хотите, чтобы текст отображался с хинтингом, встроенным в файлы шрифтов.
  • Автосохранение - используется для определения частоты автоматического сохранения изменений, внесенных Вами при редактировании.
    • Выберите один из доступных интервалов времени: Каждые 10 минут, Каждые 30 минут или Каждый час.
    • Выберите опцию Отключено, если Вы не хотите, чтобы редактор ONLYOFFICE™ Document Editor автоматически сохранял изменения.
  • Единица измерения - используется для определения единиц, которые должны использоваться на линейках и в окнах свойств для измерения параметров элементов, таких как ширина, высота, интервалы, поля и т.д. Можно выбрать опцию Сантиметр или Точка.

Чтобы сохранить внесенные изменения, нажмите кнопку Применить.
