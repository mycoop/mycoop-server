Seleccione color de fondo para un párrafo

Un color de fondo es aplicado al todo el párrafo.

Para aplicar un color de fondo a un párrafo o cambiar el color corriente,

  1. seleccione una combinación de colores de su documento de las combinaciones disponibles pulsando el icono Cambiar combinación de colores en la barra de herramientas superior
  2. meta el cursor sobre el párrafo necesario, elija unos párrafos usando el ratón o todo el texto pulsando una combinación de teclas Ctrl+A,
  3. abra la ventana con paletas de color pulsando el campo de color al lado de la leyenda Color de fondo en la derecha barra lateral

    Nota: usted también puede acceder a la ventana con paletas de color pulsando el enlace 'Mostrar ajustes avanzados' en la derecha barra lateral o seleccionando 'Ajustes avanzados de párrafo' en el menú contextual, después pasando a la sección 'Bordes y relleno' de la ventana 'Párrafo - Ajustes avanzados' y pulsando el campo de color al lado de la leyenda Color de fondo.

  4. seleccione algún color en las paletas disponibles

    • Colores de tema - los colores que corresponden a la combinación seleccionada de colores de un documento.
    • Colores estándar - el conjunto de colores predeterminado.
    • Color personalizado - pulse este enlace si en las paletas disponibles no hay color necesario. Seleccione el rango del color necesario desplazando el vertical control deslizante y establezca el color deseado arrastrando el selector de colores dentro del grande campo de color. Una vez seleccionado el color con selector, los valores RGB y sRGB apropiados serán mostrados en los campos a la derecha. Usted también puede especificar un color basándose en el modelo de color RGB introduciendo necesarios valores numéricos en los campos R, G, B red-rojo, green-verde, blue-azul) o introduzca el código hexadecimal sRGB en el campo marcado con el signo #. El color seleccionado aparece en la casilla Nuevo. Si anteriormente el objeto era rellenado con cualquier color personalizado, este color se muestra en la casilla Actual para que usted pueda comparar los colores original y modificado. Cuando el color se haya especificado, pulse el botón Añadir:

      El color personalizado se aplicará al párrafo y se añadirá a la paleta Color personalizado.

Para limpiar el color de fondo de un párrafo,

  1. meta el cursor al párrafo necesario, elija unos párrafos usando el ratón o todo el texto pulsando una combinación de teclas Ctrl+A,
  2. abra la ventana con paletas de color pulsando el campo de color al lado de leyenda Color de fondo en la derecha barra lateral
  3. seleccione el icono .
