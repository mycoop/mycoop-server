Insert page numbers

To insert page numbers into your document,

  1. click the Edit Header or Footer icon at the top toolbar,
  2. click Insert Page Number,
  3. select one of the following options:
    • To put a page number to each page of your document, select the page number position on the page.
    • To insert a page number at the current cursor position, select the To Current Position option.

To edit the page number settings,

  1. double-click the page number added,
  2. change the current parameters at the right sidebar: the position of page numbers on the page as well as relative to the top and bottom of the page. You can also check the Different first page box to apply a different page number to the very first page or in case you don't want to add any number to it at all. The Different odd and even pages box is used to insert different page numbers for odd and even pages.

To return to the document editing, double-click within the working area.
